# Tic-Tac-Toe

A minimal, aesthetic tic-tac-toe game with Google authentication.

**Live:** https://saisathwikpeddireddy.github.io/TicTacToe/

## Stack

- React + Vite + TypeScript
- Clerk (Google auth)

## Run locally

```bash
npm install
npm run dev
```

Requires a `.env.local` with your Clerk publishable key:

```
VITE_CLERK_PUBLISHABLE_KEY=pk_...
```
